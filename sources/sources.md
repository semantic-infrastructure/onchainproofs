# Sources

## Purpose
This page documents the primary references used to define proof, verification, and auditability in on-chain systems.
The focus is on verifiable claims, evidence lifecycles, and audit-ready state transitions.
This page is reference-only and non-commercial.

## Source hierarchy
1. Primary sources: protocol standards and formal specifications
2. Secondary sources: auditability and integrity frameworks
3. Excluded sources: token issuance, marketplaces, promotional material

## Curated sources

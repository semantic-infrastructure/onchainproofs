# Changelog

## v0.1 — Initial specification (2026-01-19)

Initial release of the on-chain proofs reference.

Includes:
- Definition of proof lifecycle (claims, evidence, attestations, state transitions)
- Specification of proof types and verification flow
- Introduction of threat model and auditability structures

Change discipline:
- Updates are additive or corrective
- No silent modification of definitions
- Material changes are versioned
